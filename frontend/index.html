Objetivo del juego

Llevar el robot indicado hasta el objetivo en la menor cantidad de movimientos posible.

Cómo se juega
  • Elegí cuántos movimientos creés que necesitás para llegar.
  • Luego intentá resolver el puzzle en esa cantidad exacta.
Movimiento de los robots
  • Cuando movés un robot, se desliza en línea recta hasta chocar con una pared u otro robot.
  • No podés detenerlo en medio del camino.
Rebotadores (barras)
  • Reflejan el movimiento del robot y el reflejo no se añade al contador de movimientos.
Puntaje
  • Si llegás al objetivo en la cantidad exacta de movimientos: +1 punto.
  • Si te pasás: -1 punto.
  • Si no llegás: no sumás puntos.
Modo práctica
  • Podés moverte libremente sin declarar movimientos.
  • Ideal para experimentar y aprender.
Modo fácil
  • Se agrega un robot gris.
  • En objetivos comodín, el robot gris solo sirve de apoyo, pero no para llegar al objetivo.

Controles

Cómo jugar
Seleccionar un robot
  • Tocá o hacé clic sobre un robot para seleccionarlo.
Mover un robot
  • Usá los botones de dirección para mover el robot.
En computadora
  • También podés usar las flechas del teclado para el movimiento de los robots y las teclas R, G, B, Y, W para seleccionar a los robots rojo, verde, azul, amarillo y gris respectivamente.
